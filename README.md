### Demo

main deki son commit vercel tarafından https://enpara.vercel.app/ adresine otomatik deploy olur.

### Local Kurulum

Repoyu klonladıktan sonra paketleri kurup

```
npm install
```

development sunucusunu çalıştırabilirsiniz.

```
npm run dev
```
